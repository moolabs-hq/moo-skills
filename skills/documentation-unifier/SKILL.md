# Goal

* Gap analysis - To complete the already existing documentations by identifying and highlighting gaps, asking questions to fill the gap and then filling the gaps in the documentations
* Ensuring Documentation consistency - Highlight inconsitency between documents provided. Many a times, when first documentation is written, it is with an assumption. As more clarity emerges, this assumption goes for a toss. Then the subsequent documentation sections honor the new assumption but we miss to update the orginal documentaion. You task includes finding such inconsistencies and either resolve if resolution is clear or highlight for me to take a decision.
